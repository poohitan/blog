У украинской гривны есть свой знак:

₴
            — у вас что-то такое вместо знака?

Существует он еще с 2004 года, когда Нацбанк утвердил его как официальный знак украинской гривны, но несмотря на это, он до сих пор не получил широкого распространения — все и дальше используют обычное «грн.» Даже неофициальный знак русского рубля, который пропагандируется несколькими энтузиастами с 2007 года, является более распространенным.

В том, что о существовании знака гривны большинство даже не догадывается, можно винить власть, которая ничего не делает для его распространения. А можно просто самому помогать распространять этот знак, везде используя его вместо сокращения «грн.» Неважно, вы продавец на рынке или владелец онлайн-магазина — знак гривны легко как написать от руки, так и использовать на веб-странице.

Как набрать знак гривны:

В Windows Vista, 7 или 8:

Нужно выбрать раскладку Украинский (расширенный)  и нажать Shift + ~

В Windows XP:

  1. Скачайте украинскую раскладку клавиатуры с символами Unicode
  2. Разархивируйте
  3. Запустите файл setup.exe (нужно иметь права администратора)
  4. В настройках языковой панели выберите раскладку Ukrainian Unicode
  5. Теперь вы можете набирать знак гривны комбинацией клавиш Правый Alt + Shift + 3 (или Ctrl + Alt + Shift + 3)

В Mac OS X 10.10 Yosemite:

Набирать знак гривны можно комбинацией клавиш Alt (Option) + Shift + Ф

В Mac OS X версий ниже, чем 10.10:

  1. Скачайте украинскую раскладку клавиатуры с символами Unicode
  2. Разархивируйте
  3. Файлы Ukrainian Unicode.keylayout  и Ukrainian Unicode.icns  скопируйте в /Library/Keyboard Layout  (для всех пользователей, нужны права администратора) или в ~/Library/Keyboard Layouts  (только для текущего пользователя)
  4. Перезавантажте комп'ютер (або зробіть Log out, Log in)
  5. Зайдите в System Preferences → International → Input Menu  и выберите Ukrainian Unicode
  6. Теперь вы можете набирать знак гривны комбинацией клавиш Alt (Option) + Shift + 3

В iOS:

Нужно открыть Настройки → Основные → Язык и текст → Регион  и выбрать там Україна. Теперь вы можете набирать знак гривны, открыв клавиатуру со специальными символами кнопкой .?123 и нажав там ₴.

Коды знака гривны:

HTML (десятичная система): &#8372;
HTML (шестнадцатеричная система): &#x20b4;
UTF-8 (шестнадцатеричная): 0xE2 0x82 0xB4 (e282b4)
UTF-8 (двоичная): 11100010:10000010:10110100
UTF-16 (шестнадцатеричная): 0x20B4 (20b4)
UTF-16 (десятичная): 8 372
UTF-32 (шестнадцатеричная): 0x000020B4 (20b4)
UTF-32 (десятичная): 8 372
Для использования в C/C++/Java: "\u20B4"
Для использования в Python: u"\u20B4"

Графический файл со знаком гривны:

Вы можете скачать графический файл со знаком гривны в четырех вариантах (Regular, Italic, Bold, Bold Italic). Файл доступен в следующих форматах:

  • svg, 2.62 кБ
  • eps, 597 кБ
  • png, 13.8 кБ, 513×585 пикселей


Более полную информацию о знаке гривны можно найти на украиноязычной версии страницы.
