Ukrainian hryvnia has a sign:

₴
            — have something like this instead of sign?

How to type hryvnia sign:

In Windows Vista, 7 or 8:

You need to choose Ukrainian (extended)  keyboard layout and press Shift + ~

In Windows XP:

  1. Download Ukrainian keyboard layout with Unicode symbols
  2. Extract
  3. Run setup.exe (need administrator privileges)
  4. In language settings choose Ukrainian Unicode keyboard layout
  5. Now you can type hryvnia sign with key combination RAlt + Shift + 3 (or Ctrl + Alt + Shift + 3)

In Mac OS:

  1. Download Ukrainian keyboard layout with Unicode symbols
  2. Extract
  3. Copy Ukrainian Unicode.keylayout  and Ukrainian Unicode.icns  to /Library/Keyboard Layout  (for all users, need administrator privileges) or to ~/Library/Keyboard Layouts  (only for current user)
  4. Restart computer (or Log out, Log in)
  5. Go to System Preferences → International → Input Menu  and choose Ukrainian Unicode
  6. Now you can type hryvnia sign with key combination Alt (Option) + Shift + 3
