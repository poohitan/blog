В української гривні є свій знак:

₴
            — у вас щось таке замість знаку?

Існує він ще з 2004 року, відколи Нацбанк затвердив його як офіційний знак української гривні, але незважаючи на це, він досі не набув широкого розповсюдження — усі й далі використовують звичне «грн.» Навіть неофіційний знак російського рубля, який пропагується кількома ентузіастами з 2007 року, є більш поширеним.

У тому, що про існування знаку гривні більшість українців навіть не здогадується, можна винуватити владу, яка нічого не зробила для його розповсюдження. А можна просто самому допомагати поширювати цей знак, всюди використовуючи його замість скорочення «грн.» Немає значення, чи ви продавець на ринку, чи власник онлайн-магазину — знак гривні легко як написати від руки, так і використати на веб-сторінці.

Як набрати знак гривні:

У Windows Vista, 7 або 8:

Потрібно вибрати розкладку Українська (розширена)  і натиснути Shift + ~

У Windows XP:

  1. Завантажте українську розкладку клавіатури з символами Unicode
  2. Розархівуйте
  3. Запустіть файл setup.exe (потрібно мати права адміністратора)
  4. В налаштуваннях мовної панелі виберіть розкладку Ukrainian Unicode
  5. Тепер ви можете набирати знак гривні комбінацією клавіш Правий Alt + Shift + 3 (або Ctrl + Alt + Shift + 3)

У Mac OS X 10.10 Yosemite:

Набирати знак гривні можна комбінацією клавіш Alt (Option) + Shift + Ф

У Mac OS X нижчих версій, ніж 10.10:

  1. Завантажте українську розкладку клавіатури з символами Unicode
  2. Розархівуйте
  3. Файли Ukrainian Unicode.keylayout  та Ukrainian Unicode.icns  скопіюйте в /Library/Keyboard Layout  (для усіх користувачів, потрібні права адміністратора) або в ~/Library/Keyboard Layouts  (лише для поточного користувача)
  4. Перезавантажте комп'ютер (або зробіть Log out, Log in)
  5. Зайдіть в System Preferences → International → Input Menu  і оберіть Ukrainian Unicode
  6. Тепер ви можете набирати знак гривні комбінацією клавіш Alt (Option) + Shift + 3

У iOS:

Потрібно відкрити Параметри → Загальні → Міжнародні → Місцевий формат  та вибрати там Україна. Тепер ви можете набирати знак гривні, відкривши клавіатуру зі спецільними символами кнопкою .?123 та натиснувши там ₴.

У Android:

Завантажте і встановіть додаток SwiftKey Keyboard. У додатку в списку Доступні мови виберіть Українська. Після завантаження українського мовного пакету, на клавіатурі з’явиться кнопка ₴. Якщо такої кнопки не буде, але буде кнопка з символом іншої валюти, наприклад, $ — натисніть на неї та потримайте, після цього має з’явитись список з доступними символами валют. Серед них виберіть символ гривні.

Коди знаку гривні:

HTML (десяткова система): &#8372;
HTML (шістнадцяткова система): &#x20b4;
UTF-8 (шістнадцяткова): 0xE2 0x82 0xB4 (e282b4)
UTF-8 (двійкова): 11100010:10000010:10110100
UTF-16 (шістнадцяткова): 0x20B4 (20b4)
UTF-16 (десяткова): 8 372
UTF-32 (шістнадцяткова): 0x000020B4 (20b4)
UTF-32 (десяткова): 8 372
Для використання у C/C++/Java: "\u20B4"
Для використання у Python: u"\u20B4"

Графічний файл зі знаком гривні:

Ви можете завантажити графічний файл зі знаком гривні у чотирьох варіантах (Regular, Italic, Bold, Bold Italic). Файл доступний у наступних форматах:

  • svg, 2.62 кБ
  • eps, 597 кБ
  • png, 13.8 кБ, 513×585 пікселів

Правило використання знаку:

Знак гривні потрібно писати після ціни, а не перед, і через пропуск. Приклади:

Правильно:
Неправильно:
100 ₴
100₴
₴100
